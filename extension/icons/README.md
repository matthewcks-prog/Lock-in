# Lock-in Extension Icons

Placeholder icons for the Lock-in Chrome extension.

## Required Sizes

- icon16.png (16x16) - Toolbar icon
- icon48.png (48x48) - Extension management page
- icon128.png (128x128) - Chrome Web Store

## How to Replace These Placeholders

1. Create your icon designs with the following specs:

   - Simple, recognizable design
   - Works well at small sizes
   - Suggested theme: Lock/Padlock + Book/Learning symbol
   - Use the brand colors: #667eea and #764ba2

2. Generate PNG files at the required sizes (16x16, 48x48, 128x128)

3. Replace the placeholder files in this directory

4. Tools for creating icons:
   - Figma (https://figma.com)
   - Canva (https://canva.com)
   - GIMP (https://gimp.org)
   - Online: https://favicon.io or https://realfavicongenerator.net
